# License To Be Selected

The authors should select a formal open-source license before making this repository public.

Common choices:

- MIT: permissive, short, common for academic software.
- BSD-3-Clause: permissive with explicit non-endorsement language.
- Apache-2.0: permissive with explicit patent language.
- GPL-3.0: copyleft; derivative software must remain GPL-compatible.

For a private reviewer repository, this placeholder is acceptable. For a public GitHub/Zenodo release, replace this file with the chosen license text and update `CITATION.cff`.
